# python-basic-skeleton

## 📝 **Description**
This is a simple project with Docker, Python, uv, lint(ruff) and env manager(pydantic).

## 🛠 **Dependencies**
- Docker
- Docker Compose

## 🚀 **Running**
1. Build the Docker image:
    ```sh
    make build
    ```
2. Run the app:
    ```sh
    make run
    ```
